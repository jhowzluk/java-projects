# Java Projects [![Java](https://skillicons.dev/icons?i=java&theme=light)](https://skillicons.dev)

Bem-vindo ao meu repositório de estudos em **Java**! 

Este espaço é dedicado a centralizar meus projetos práticos, exercícios e anotações enquanto aprofundo meus conhecimentos no ecossistema Java. O objetivo principal aqui é documentar minha evolução no desenvolvimento backend, desde os fundamentos da linguagem até a exploração de frameworks como o Spring.

## Objetivos de Aprendizado

- Consolidar a base de Programação Orientada a Objetos (POO).
- Entender estruturas de dados e algoritmos aplicados em Java.
- Praticar a criação de APIs e serviços backend.
